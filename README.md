# Da Hood Script

A clean Da Hood script using Linoria Library.

## Features
- 💰 **Cash Aura** - Auto collects nearby dropped cash
- 💸 **Cash Drop** - Drop $15k or $5k from wallet
- 👁️ **Player ESP** - See players through walls (no lag)
- 📛 **Name Tags** - Shows player names with health

## Usage

```lua
loadstring(game:HttpGet('https://raw.githubusercontent.com/Cosmic4796/DaHoodScript/main/Loader.lua'))()
```

## Controls
- Press `End` to toggle menu
